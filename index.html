Do I need to know how to code?

No. SlickCMS is a blog designed to be set up and running in 30 seconds by anyone. The only thing you'll need to learn is how to upload files to your server (here's how to do it in 5 minutes: FTP guide).

You will also want to check out how Markdown works so you can add images, titles, bold text and make your site look pretty and fancy (You can write HTML too).

How does it work?

You create files such as this:

Hello, my name is **Xavi** and I am from Barcelona.

And SlickCMS automatically generates everything, learn more.

Technical details

SlickCMS is a database-less blogging system that auto-generates static files which means pages are loaded really really fast, faster than Wordpress, Joomla, Blogger and 99% of any other content management and blog systems you can think about. It is a great option for cheap servers and old computers. And also if you want to run hundreds of blogs without compromising your CPU.
